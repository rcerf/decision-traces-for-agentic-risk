# Strategic Risk Assessment Product

Status: portfolio-grade prototype  
Purpose: generate a public-signal risk assessment that can be shared as the primary artifact for the OpenAI Agentic Risk Analyst application.

## Product Thesis

The strongest shareable artifact is not only the framework. It is a decision-ready risk assessment produced by the framework.

This module ranks public-signal hypotheses for agentic AI risks that may be under-owned, under-measured, or under-instrumented from the outside looking in. It does not claim that OpenAI has missed these risks. It asks which risks are worth testing for coverage given public evidence.

## Inputs

- `data/public_signal_risk_hypotheses.json`

Each hypothesis includes:

- likelihood
- impact
- novelty
- actionability
- affected surfaces
- possible under-instrumentation reason
- early indicators
- safe probe
- decision needed
- public sources

## Output

- `reports/agentic_risk_assessment_public_signals.md`

The report is generated by:

```bash
python3 studies/strategic-risk-assessment/src/generate_assessment.py --output reports/agentic_risk_assessment_public_signals.md
```

## Intended Reader

- Reviewers who own a current-risk operating picture for agentic systems.
- Any strategic-intelligence or risk-operations reviewer deciding whether this artifact shows useful operating judgment.

## Caveat

The report uses public information only. It should be read as a structured set of testable hypotheses, not as an external claim about OpenAI's private risk register.
